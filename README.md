# 🏡 Real Estate Booking Management System

A **Django-based Real Estate Booking Management System** that enables users to search, view, and book rental properties online while allowing realtors to manage property listings and customer enquiries.

---

## 📌 Project Overview

The Real Estate Booking Management System is a web application developed using **Python Django**, **HTML**, **CSS**, **Bootstrap**, and **SQLite**. It provides a digital platform where users can browse available properties, view detailed information, schedule visits, make bookings, and generate rental agreements.

The system also allows realtors to add and manage properties while administrators can monitor the overall platform.

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sharanya200224/Real-Estate-Booking-Management-System.git
```

Go to the project folder:

```bash
cd Real-Estate-Booking-Management-System
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux/Mac**

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run database migrations:

```bash
python manage.py migrate
```

Start the server:

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
